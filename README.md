## 介绍

出于对消息队列的兴趣，以及为了更好的学习[rocketmq](https://github.com/apache/rocketmq)和[kafak](https://github.com/apache/kafka)，我选择对mq的核心功能进行了实现。基于这些开源mq的架构和设计思想，我尝试用自己的方式进行了实现，该项目也主要是出于**个人学习目的**

目前已实现如下功能：

+ 生产者以同步、异步、单向方式发送消息
+ 单个消息和批量消息
+ 发送普通消息、顺序消息、延时消息
+ 超时检测和失败重试
+ 钩子函数
+ 发送队列负载均衡
+ 消费者订阅主题
+ pull和push方式获取消息
+ 指定消息拉取
+ 批量拉取
+ 消费者组管理与监听
+ 消费失败重试
+ 广播消费模式和集群消费模式
+ 本地位移管理
+ broker端持久化存储消息
+ 管理消费者位移
+ 管理生产者和消费者，心跳机制
+ 向注册中心注册
+ 提供重试队列、延时队列和死信队列
+ 多种注册中心
+ 与spring框架集成
